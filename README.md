# Angelica Ayos — Portfolio

Personal portfolio website built to showcase my work and signal my transition from Senior Frontend Developer to Full Stack Developer.

## Features
- LLM-powered chat (in progress)
- Spotify API integration (in progress)
- Contact form with email + file attachment
- Light/dark mode toggle
- WCAG compliant
- Warm autumn color palette

## Tech Stack

**Frontend**
- Next.js 16, React, TypeScript, Tailwind CSS v4

**Backend**
- Java 21, Spring Boot 3.5, Maven

**Infrastructure (planned)**
- AWS, Terraform, Docker, Redis

## Running Locally

**Frontend**
cd frontend
npm install
npm run dev

**Backend**
cd backend
mvn spring-boot:run
